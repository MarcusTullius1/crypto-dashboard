# Crypto Dashboard

**Mini Crypto Dashboard built with React and Django**  
This is a learning project in an intermediate state. The dashboard tracks a real Bitcoin wallet balance, transactions, and cryptocurrency news using live APIs.

Crypto Dashboard allows users to:

- Maintain portfolio data in **localStorage** for persistent state
